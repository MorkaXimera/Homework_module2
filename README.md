# Домашняя работа по модулю 2

## Виджет банковских операций клиента
1. Функция get_mask_card_number возвращает маску номера карты
2. Функция get_mask_account возвращает маску номера счёта
3. Функция mask_account_card возвращает маску номера счета или карты
4. Функция get_date возвращает дату в формате ДД.ММ.ГГГГ
5. Функция filter_by_state фильтрует список словарей по ключу state
6. Функция sort_by_date сортирует список словарей по дате
7. Функция filter_by_currency возвращает итератор, который поочередно выдает транзакции, где валюта операции соответствует заданной
8. Функция transaction_descriptions возвращает описание каждой операции
9. Генератор card_number_generator выдает номера банковских карт в формате XXXX XXXX XXXX XXXX
10. Декоратор, который логирует начало и конец выполнения функции,
    ее результаты или возникшие ошибки
11. Функция json_read принимает на вход путь до JSON-файла и возвращает список словарей с данными о финансовых транзакциях
12. Функция get_transactions_list считывает информацию о транзакциях из объекта json
13. Функция get_transaction_amount принимает на вход транзакцию и возвращает сумму транзакции
14. Функция csv_reader позволяет прочесть транзакции из файла в формате .csv и возвращает список словарей
15. Функция xlsx_reader позволяет прочесть транзакции из файла в формате .xlsx и возвращает список словарей
16. Функция process_bank_search принимает список словарей с данными о банковских операциях и
    строку поиска, и возвращает список словарей, у которых в описании есть данная строка
17. Функция process_bank_operations принимает список словарей с данными о банковских операциях и список категорий операций,
    и возвращает словарь, в котором ключи — названия категорий,
    а значения — количество операций в каждой категории


## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/MorkaXimera/Homework_2.git
```
2. Установите зависимости:
```
pip install -r requirements.txt
```
